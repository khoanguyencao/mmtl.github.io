Research Projects

We are currently working on the following projects/questions:

1) Towards Medical AGI
Reasoning models have demonstrated superhuman performance in verifiable domains, i.e. domains with explicitly correct answers. This project aims to build a clinical reasoning model which outperforms the vast majority of physicians, via a combination of step-by-step and reinforcement learning methods.

2) Medical AGI Benchmarking
Few benchmarks exist to measure medical AGI's performance against doctors. This project aims to develop comprehensive benchmarks which allow us to definitively measure the broader field's progress towards medical AGI.

3) Clarity: AI-Assisted Clinical Guidelines
Guidelines are poorly utilised in hospitals. This project is a collaboration with Monash Health to build a retrieval-augmented generation system which provides hospital staff with AI-assisted access to both clinical and non-clinical guidelines.

4) Chute: AI-Assisted Quality Management & Registries
Significant effort is required for quality management tracking and contribution to registries. This project is a collaboration with Alfred Health to build an AI system which automatically summarises quality management metrics and registry submissions from EMR data.
